## User Stories

### Sprint 1
- As a librarian, I want to add books to the system so that the library inventory can be maintained.
- As a librarian, I want to prevent duplicate book IDs so that each book is uniquely identifiable.

### Sprint 2
- As a librarian, I want to allow users to borrow books so that books can be issued to readers.
- As a librarian, I want to prevent users from borrowing unavailable books so that the system maintains correct availability status.
- As a librarian, I want to allow users to return borrowed books so that returned books become available again.

### Sprint 3
- As a librarian, I want to generate library reports so that I can view the current status of books in the system.
